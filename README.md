## Getting Started

### Prerequisites

Install the server and run the server.

- npm
  ```sh
  cd graphql-endpoint
  ```
- yarn
  ```sh
  yarn install
  ```
- start server
  ```sh
  yarn start
  ```

### Scandiweb Client

Ensure the server is running and start the client.

1. clone the repo
   ```sh
   git clone https://github.com/kmabika/minimal-store-scandiweb
   ```
2. install packages
   ```sh
   yarn install
   ```
3. start client
   ```sh
   yarn start
   ```
